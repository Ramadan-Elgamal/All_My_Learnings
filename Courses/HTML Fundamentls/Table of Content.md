### What is HTML about?
`HTML` stands for `HyperText Markup Language`.

But what hyper or Markup means exactly?

What differ hyper from just plain text is the connections between the text like for example, you can click on a word and jumb to completely different page or website or other sections in the same page you are now on.

So what Markup language means?

Markup language is type of languages that its main purpose is describe how the browser reads the content of your page. It is the way to define the structure of the DOM.

Let's leave the DOM concept for other video in the beyond playlist.

Okay, how does `HTML` work then?
In `HTML` you can use tags by opening `<>` brackets and put the type of tag you want to use.
Based on the structure, tags is divided into two types ( paired tags / Self-closing tags).
Paired tags is when start the tag by `<>` and close it also by `</>`. You can name this tags as container tags as it is used to act as a parent for other tags. For example, you can put `input` tag in `form` tag to ask the user to enter his name or email.

For the other type ( Self-closing tags):
It is when you just open the tag `< >` and just add `/` in it. These tags don't contain other tags. They jus insert something into the page. For example the `img` tag to insert an image or the `br` tag to insert a breaking line between the section of the webpage.
